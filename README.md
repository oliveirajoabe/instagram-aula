# Aula de Clone do layout do Instagram

- Esta aula que foi feita atraves da [Digital Innovation One](https://digitalinnovation.one/), se refere aos principios basicos de flex box, onde com a utilização do mesmo realizamos um clone do layout de login do instagram, de forma que seja totalmente responsivo.
- Esta aula se trata de somente pratica de front end, utilizando flex box, portanto não contem interações
